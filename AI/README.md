# AI

## What is AI?
Artificial Intelligence (AI) is the simulation of human intelligence in machines that are programmed to think and learn.

## Architecture
- Data Collection: Gathering and preparing data.
- Model Training: Building machine learning models.
- Inference: Making predictions with trained models.

## Example (Python)
```python
from sklearn.linear_model import LinearRegression

model.fit(X_train, y_train)
```
